# Swarms Weekly Recap [July 13-20]: Fiat Payments on Swarms Marketplace, 7,000 GitHub Stars, Swarms Academy

The past week moved across nearly every layer of the Swarms stack: education, the open-source framework, the Marketplace payment rail, the cloud platform, and the longer-term thesis behind all of it. This post collects the five updates in one place, with a short summary of each and links to the full announcement behind it.

## Swarms Academy Is Live

![Swarms Academy](/academy_picture.png)

We launched Swarms Academy, a free learning platform for developers building production AI agents and multi-agent systems. The first course available is the Swarms API Course, a curriculum of 31 lessons across four production-focused modules. It covers the full workflow from a first authenticated request through single-agent capabilities like streaming, structured outputs, tool calling, and MCP integration, and on to multi-agent orchestration and production concerns. The course includes live API exercises you run in the page, hands-on projects, quizzes, and progress tracking, so a developer can move from their first agent completion to a deployed multi-agent system without leaving the lesson. 📚

Read the full announcement: [Introducing Swarms Academy](/blog/introducing-swarms-academy)

## Swarms Reaches 7,000 GitHub Stars

![Swarms reaches 7,000 GitHub stars](https://pbs.twimg.com/media/HNdR91uXoAAwmHK?format=jpg&name=large)

Swarms crossed 7,000 stars on GitHub, four years after it started as a research project. In that time it has grown into an open-source platform used by thousands of developers building multi-agent systems in production. The milestone reflects the contributors, users, and researchers who have filed issues, opened pull requests, and shaped the framework over four years of development. It marks a point of maturity for the project rather than a finish line.

Read the full announcement: [Swarms Reaches 7,000 GitHub Stars](/blog/swarms-7000-github-stars)

## Fiat Payments Are Live on the Swarms Marketplace

![Fiat payments live on the Swarms Marketplace, powered by Stripe](/swarms_blog_fiat_cover.png)

Fiat payments are now live on the Swarms Marketplace, powered by Stripe. Buyers in more than 100 countries can purchase AI agents and prompts using familiar methods including bank cards, Apple Pay, Google Pay, Alipay, and Klarna, across 100+ currencies. The system is built on Stripe Connect: sellers connect an account once, receive automatic payouts to their bank, and keep 90% of every sale under the same 10% marketplace fee. Buyers get instant, lifetime access after purchase, with no wallet or manual withdrawal step. Existing crypto payment support remains fully in place, so the two options run side by side from a single seller account.

Read the full announcement: [The Agent Economy Now Takes Card, Powered by Stripe](/blog/fiat-payments-live-swarms-marketplace)

## Two Months of Swarms Cloud Updates

![Swarms Cloud changelog: two months of platform updates](/swarms_cloud_2month.png)

We published a comprehensive changelog covering everything shipped across Swarms Cloud over the past two months. Rather than highlighting only the major launches, it provides a day-by-day timeline of every feature, improvement, performance optimization, and bug fix that reached production. Highlights include the new Workflow Builder for visually orchestrating multi-agent systems, a public model catalog spanning more than 1,500 models, a Token Usage dashboard with real-time cost and usage analytics, guided onboarding with $5 in free API credits for new users, and dozens of reliability enhancements and UI improvements.

Read the full changelog: [Swarms Cloud Changelog: Two Months of Platform Updates](/blog/swarms-cloud-changelog)

## The Autonomous Economy: Founders Keynote

![Founders keynote on the autonomous economy at AI Salon](/autonomous_economy_keynote.png)

Swarms co-founder and CEO Kye Gomez presented a keynote at the AI Salon to an audience of more than 80 people on the future of business: corporations operated largely by AI agents, scaling through software rather than headcount. The argument is that as the cost of intelligence falls, a new operating model becomes viable, where companies execute continuously with minimal overhead and expand capability by orchestrating intelligent systems instead of hiring. The keynote frames the infrastructure Swarms builds as the foundation for that shift.

Read the full announcement: [The Autonomous Economy Is Coming: Our Founders Keynote at AI Salon](/blog/autonomous-economy-keynote)

## Conclusion

Taken together, these updates trace a single arc. Swarms Academy lowers the barrier to learning the stack, the 7,000-star milestone reflects four years of framework maturity, fiat payments open the Marketplace to a global base of buyers and sellers, the Cloud changelog documents steady platform progress, and the founders keynote sets the direction the rest of the work points toward. Each piece supports the same goal: making it practical to build, deploy, and scale production multi-agent systems. We are just getting started.
