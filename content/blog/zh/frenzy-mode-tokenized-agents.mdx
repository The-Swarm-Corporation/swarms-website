# Frenzy Mode 深度解析：如何以 2 倍创作者费用发行代币化智能体

[Swarms 市场](https://swarms.world)上的代币化智能体或提示词，本质上是一个普通的市场列表，只不过在 Solana 上附加了一个可实时交易的代币。创作者不再是每个买家只付费一次，而是从这枚代币的每一笔交易中赚取费用，只要还有人在交易，收入就会持续。如果你想了解完整机制，包括绑定曲线（Bonding Curve）、市值检查点以及完整的发布流程，请先阅读 [《代币化智能体：它是什么、如何发布，以及创作者如何获得持续收益》](https://swarms.world/blog/tokenized-agents-launch-revenue)。

本文将深入探讨该流程中的一个具体开关：Frenzy Mode。它会改变你的代币费用结构，也会改变代币在市场中的展示方式，因此在发布时决定是否打开它之前，值得把它彻底搞清楚。

## Frenzy Mode 到底做了什么

默认情况下，Swarms 上每一个代币化列表的总交易费用为每笔交易的 1.0%，平均分配：0.5% 归创作者，0.5% 归 Swarms 平台。这是任何代币化智能体或提示词的基础经济模型。

Frenzy Mode 是发布时可选的一个开关，位于代码符号（Ticker）和计价货币字段旁边，仅适用于智能体和提示词（工具完全不支持代币化，无论是否启用 Frenzy）。启用它并不只是在创作者和平台之间重新分配原有的 1.0%，而是把这枚代币整体路由到一个不同的费用档位。标准流程调用 Jupiter 合作伙伴 API 时使用一个 API 密钥；Frenzy 模式调用同一 API 时使用第二个专属密钥，在代码库中被引用为 `JUPITER_API_KEY_TWO_PERCENT_FEE`。发布 API 自身的文档注释说明了这个密钥的含义：“frenzy” 使用 2 倍费用的 Jupiter API 密钥，并将代币列入 Frenzy 排行榜。

这是全文最需要精确对待的核心事实：Frenzy Mode 不只是把现有费用中创作者的那一份翻倍，而是把每一笔交易被收取的总费用直接翻倍。应用内展示给创作者、供其决定是否启用的文案原文是：“Earn 2x fees on all volume. You receive 1% of all buy and sell volume.”（获得 2 倍交易量费用。你将获得所有买卖交易量的 1%。）这句话在发布页面、代币化弹窗和批量发布流程中完全一致，它直接确认了创作者的份额：所有买卖交易量的 1%，是标准 0.5% 的两倍。既然 Frenzy 模式下的总费用是 2%（对应“两个百分点费用”这个 API 密钥名称），而创作者声明的份额是 1%，那么剩下的 1% 归 Swarms 平台，同样是其标准 0.5% 份额的两倍。平台一侧的这个数字并没有在代码中以字面字符串的形式出现，它是从声明的总费用中减去声明的创作者份额直接推导出来的。

| | 标准模式 | Frenzy Mode |
| --- | --- | --- |
| 每笔交易总费用 | 1.0% | 2.0% |
| 创作者份额 | 0.5% | 1.0% |
| 平台份额 | 0.5% | 1.0% |
| 适用范围 | 智能体与提示词 | 智能体与提示词 |

代币化的其他部分不受任何影响。绑定曲线、市值检查点、一次性的 0.04 SOL 发布费用、SOL 与 USDC 之间的计价货币选择：无论是否启用 Frenzy，这些机制的运作方式完全相同。Frenzy 只改变交易费用本身及其去向。

## 如何在市场中识别 Frenzy 代币

每一个代币化列表都存有一个 `fee_selection` 字段，如果发布时打开了该开关，其值就是 `"frenzy"`。这个字段决定了买家能看到的一切：

- 在[注册表（Registry）](https://swarms.world/platform/registry)中，Frenzy 代币的列表卡片带有一个带动画效果的橙色火焰徽章，与 Vault Mode 使用的靛蓝色锁形徽章区分开来。
- 注册表的定价筛选器中包含“Frenzy”这一选项，与免费、付费、代币化、Vault 并列，买家可以据此只筛选出 Frenzy 模式的列表。同样的选项也存在于排除筛选中，因此不想承担更高往返交易费用的买家，也可以直接把 Frenzy 代币排除在外。
- 市场探索首页运行着一个专门的“Top Frenzy Agents & Prompts”（顶级 Frenzy 智能体与提示词）轮播板块，副标题为“以 Frenzy Mode 发布的智能体与提示词”，数据来自 `/api/frenzy-tokens` 接口，该接口查询所有 `fee_selection` 为 `'frenzy'` 的智能体与提示词，并按 Jupiter 提供的实时市值排序。该轮播板块中的“查看全部”链接会直接跳转到已按 `pricing=frenzy` 预筛选的注册表页面。

Frenzy 并非一个只影响你自己费用账单的隐藏设置。它是一个可见、可筛选、可浏览的属性，在意往返交易成本的买家有直接明了的方式在交易前就能识别出来。

## 如何发布启用 Frenzy Mode 的代币化智能体或提示词

1. 前往[发布页面](https://swarms.world/launch)，像平常一样填写你的列表：名称、描述、内容，以及发布普通智能体或提示词所需的其他字段。
2. 在定价（Pricing）区域打开代币化（Tokenization）开关（仅适用于智能体与提示词）。
3. 设置代码符号（Ticker Name，最多 10 位大写字母和数字），并选择计价货币（Quote Mint）：SOL 或 USDC。
4. 在同一个代币化面板中，打开 Frenzy Mode 开关：一张带橙色边框和火焰图标的卡片，标注为“Frenzy Mode”，下方文案为“Earn 2x fees on all volume. You receive 1% of all buy and sell volume.”
5. Vault Mode 就在 Frenzy Mode 正下方，作为另一个独立的开关。两者互不干扰，你不需要二选一，同时开启两者（即一个既有持有门槛、又收取双倍 Frenzy 费用的代币）也是该流程支持的合法组合。
6. 连接一个 Solana 钱包。任何代币化列表都需要连接钱包，因为代币铸造是发布动作的一部分。
7. 点击发布，并在提示时支付一次性的 0.04 SOL 发布费用。你的列表随即上线，其代币带有 `frenzy` 费用标识、橙色火焰徽章，并从第一笔交易起就适用翻倍的 2% 交易费用。

无论是 Frenzy 还是标准模式，创作者费用都会实时在链上累积，领取方式也完全一致：在账户设置的 Creator Fees（创作者费用）标签页连接钱包，即可查看每个代币未领取和已领取的费用，并通过你自己签署的交易进行领取。

## 权衡取舍：Frenzy Mode 值得启用吗

无论是发布流程还是官方文档，都没有在这一点上给出明确建议，因此以下内容属于推理分析，而非产品方的既定结论，而且这个权衡确实是双向的。

2% 的总往返费用是标准列表的两倍，而交易者在比较两个类似代币时，可以通过注册表的 Frenzy 筛选功能直接看出这个差异。对费用敏感的交易者，在存在一个非 Frenzy 的类似列表时，可能会直接选择费用更低的 1% 标准档位，这对预期交易量而言是实实在在的损失。

反过来看，启用 Frenzy Mode 的创作者从第一笔交易开始，就能在每一笔实际发生的交易中获得双倍收入。对于一个你预计无论如何都会获得关注的列表，比如一个已经小有名气的智能体、一个正在被积极推广的提示词，或者拥有一批因真实兴趣而非费用套利去交易它的粉丝群体，把收益翻倍就是一个简单直接的收益。Frenzy Mode 还会把你的列表展示给一批专属受众：Frenzy 轮播板块和注册表筛选器正是为那些愿意接受、甚至主动寻找更高费用代币的买家而存在，这是标准代币化列表所没有的另一条曝光渠道。

简而言之，Frenzy Mode 奖励的是对需求的信心。如果你不确定一个列表能否带来真实的交易量，标准的 1% 分成是风险更低的默认选项，因为更低的费用意味着边际交易者少一个转身离开的理由。如果你确信无论费用高低，人们都会交易它，那么 Frenzy Mode 就是一条直接的路径，让你从这些活动中赚取的收益翻倍。

## 常见问题

**标准模式和 Frenzy Mode 之间的确切费用差异是多少？**
标准列表每笔交易收取 1.0% 的总费用，0.5% 归创作者，0.5% 归平台。Frenzy Mode 每笔交易收取 2.0% 的总费用，1.0% 归创作者，1.0% 归平台。创作者的收益翻倍，交易者承担的总成本也随之翻倍。

**我可以在同一个列表上同时启用 Frenzy Mode 和 Vault Mode 吗？**
可以。两者是相互独立的开关，同时启用会产生一个既有代币持有门槛（只有持有者才能使用）、又在每笔交易中收取双倍 Frenzy 费用的列表。

**工具（Tools）可以使用 Frenzy Mode 吗？**
不可以。代币化本身只对智能体和提示词开放，因此作为代币化附加项的 Frenzy Mode，对工具同样不可用。

**买家在交易前如何知道某个代币启用了 Frenzy Mode？**
Frenzy 代币的市场卡片上带有橙色火焰徽章，注册表的定价筛选器中也有专门的“Frenzy”选项。探索首页还运行着一个按实时市值排序的“Top Frenzy Agents & Prompts”轮播板块，并附有跳转到已预筛选注册表页面的链接。

**我能在代币已经发布之后再开启 Frenzy Mode 吗？**
该开关是在代币铸造这一链上操作的同一时刻、也就是发布时设定的。本文讨论的是发布时的流程；如果你需要修改一个已经上线代币的费用设置，请查看账户中的 Creator Fees 标签页，或联系支持团队。
