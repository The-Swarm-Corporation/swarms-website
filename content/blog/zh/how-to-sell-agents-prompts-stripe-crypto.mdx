# 如何在 Swarms Marketplace 上出售你的智能体与提示词：Stripe 与加密货币收款完全指南

如果你构建了一个值得复用的智能体、系统提示词或工具，[Swarms Marketplace](https://swarms.world) 就是把这份工作变成收入的地方。买家在[注册表](https://swarms.world/platform/registry)中浏览商品，支付你设定的一次性价格，立即获得访问权限。无论买家用银行卡还是加密货币付款，你都能拿到每笔销售额的 90%。

这篇指南按实际操作顺序，逐步讲解整个流程：设置收款方式、为商品定价、发布商品，以及买家下单那一刻究竟发生了什么。这里描述的每一个机制，都是 Marketplace 今天真实的运作方式，而不是某种未来的设想。

## 为什么要在 Marketplace 上出售

Swarms Marketplace 的存在，是为了让构建智能体组件的人，与需要这些组件、而不想从零开始自建的团队相互连接。如果你已经写出了一个扎实的研究智能体、一条调校精良的系统提示词，或者一个可复用的工具函数，发布它不花一分钱，而收益是直接的：每笔销售的 90% 会直接进入你的账户。没有上架费，没有订阅套餐，而且截至目前，发布付费商品之前，并不需要达到任何最低已发布商品数量或星级评分要求。任何账户都可以上架付费商品。

## 第一步：设置 Stripe 卖家账户（如果你想接受银行卡付款）

如果你希望买家能够用银行卡付款，请先完成这一步。对于只想做加密货币销售的卖家来说，这一步完全是可选的，但它只需要几分钟，却能为你打开最大规模的潜在买家群体，毕竟不是所有人都持有 SOL。

1. 打开账户设置中的[卖家标签页](https://swarms.world/platform/account?tab=seller)。
2. 在下拉菜单中选择你所在的国家。
3. 点击 **Set up seller account**。

这次点击会为你创建一个 Stripe Express 关联账户，并将你重定向到 Stripe 自己托管的入驻流程，在那里你需要提供收款信息、身份信息，以及 Stripe 验证你所需的其他材料（标准 KYC 流程）。如果中途被打断，进度会被保存，回到卖家标签页即可从中断的地方继续。

仅仅填完入驻表单，并不代表你的账户已经激活。只有当 Stripe 反馈你的转账能力已经生效、打款功能已经启用时，状态才会变为激活。对大多数卖家来说，这个过程通常很快，不过 Stripe 自身的验证时长会因国家和风险等级而有所不同。账户激活后，同一个卖家标签页会提供一键入口，直接进入你的 Stripe Express 控制台，随时查看余额和打款历史。

如果你只打算用 SOL 销售，完全可以跳过这一步。加密货币收款完全不经过 Stripe。

## 第二步：确定定价，选择智能体、提示词还是工具

Marketplace 支持三种类型的商品，三者都可以作为付费商品出售：

| 类型 | 包含内容 | 典型用途 |
| --- | --- | --- |
| **智能体（Agent）** | 可执行代码、依赖项、环境变量 | 多步骤工作流、集成、编排式分析 |
| **提示词（Prompt）** | 仅包含自然语言指令 | 人设、任务指令、结构化输出模板 |
| **工具（Tool）** | 一个带类型标注的 Python 函数 | 被其他智能体调用的单一功能操作 |

选择与你实际构建内容相匹配的类型，然后以美元设定一次性价格。Marketplace 上没有订阅：买家一次性付款，即可获得内容的完整、永久访问权限，无论那是完整的提示词文本，还是智能体的代码与配置。

定价思路可以参考任何可复用软件的定价方式：想一想买家如果要自己构建出等效的东西，需要花多少时间或专业知识，然后把价格定在这个成本之下。

## 第三步：从 /launch 发布，并选择你的支付通道

前往[发布页面](https://swarms.world/launch)，按照你平时的方式发布：命名、撰写描述、附上代码或提示词内容，并标记为 **Paid**。

在定价部分，你会看到一个问题：**How do you want to get paid?（你想如何收款？）**，有两个选项：

- **Crypto (SOL)**：收入打入你注册的 Solana 钱包
- **Card / Fiat (Stripe)**：收入打入你已连接的 Stripe 账户

这个选择是按商品逐一进行的，而不是按账户。你完全可以让一个智能体走加密货币销售，另一个提示词走法币销售，两者用的是同一个个人资料，平台手续费在两条通道下也完全相同。

如果你选择 Card / Fiat，但你的 Stripe 账户尚未激活（转账能力尚未启用），发布页面会当场阻止发布，并显示一条提示，指引你回到卖家标签页先完成入驻。完成之后回来，阻拦就会解除。

有一点无论你怎么选都是固定的：代币化发布始终以加密货币结算。法币（Stripe）只适用于直接的、非代币化的付费商品。

## 第四步：买家下单后会发生什么

这是两条支付通道真正分道扬镳的地方，因为资金流经的是完全不同的基础设施。

**加密货币（SOL）。** 买家的钱包会签署一笔 Solana 交易，其中一次性包含两条转账指令：一条指令将 10% 的平台手续费发送到财库钱包，另一条指令将你的 90% 份额直接发送到你注册的钱包。这两笔转账与购买行为本身处于同一笔交易之中。中间没有任何第三方托管你的资金，也没有单独的打款步骤。结算是即时的、非托管的。

**银行卡 / 法币（Stripe）。** 买家通过 Stripe 托管的结账页面付款。Stripe 自动完成资金拆分：你的 90% 份额作为底层收款上的应用费拆分，进入你的 Stripe Express 账户余额，同一步骤中，10% 的平台手续费被收取。此后，Stripe 会按照自己的正常打款周期，把你账户余额中的钱打入你绑定的银行账户，这与任何使用 Stripe Connect 的企业所遵循的周期相同。这个周期会因国家和账户风险状况而不同，因此应把它理解为"Stripe 的标准打款时长"，而不是一个固定的天数。

| | 加密货币（SOL） | 银行卡 / 法币（Stripe） |
| --- | --- | --- |
| 资金落地位置 | 你的 Solana 钱包 | 你的 Stripe Express 余额 |
| 到账时间 | 即时，在购买交易中完成 | 按 Stripe 的标准打款周期 |
| 谁来划转资金 | 买家自己签署的交易 | 作为支付处理方的 Stripe |
| 平台手续费 | 10% | 10% |
| 你的所得 | 90% | 90% |

## 费用结构

这项费用是固定的，不会因为支付通道、你的账户历史或商品价格而有任何变化。Swarms 对每笔完成的销售收取 10%，你获得 90%。

| 售价 | 平台手续费（10%） | 你的所得 |
| ---------- | ------------------ | ----------- |
| $5.00 | $0.50 | $4.50 |
| $20.00 | $2.00 | $18.00 |
| $100.00 | $10.00 | $90.00 |

没有上架费，没有订阅套餐，也不会因为销量或评分而有任何浮动费率。只有你赚到钱，Marketplace 才有收入。

## 常见问题

**我需要达到最低发布数量或评分才能出售付费商品吗？**
不需要。截至目前，在 Marketplace 上发布付费商品之前，并不存在任何最低已发布商品数量或最低评分要求。任何账户都可以上架一件商品。

**我卖东西一定需要 Stripe 账户吗？**
不一定。只有当你想接受银行卡付款时才需要 Stripe。如果你只想用 SOL 销售，完全可以跳过卖家标签页，直接在发布时选择 Crypto (SOL)。

**我可以有些商品用加密货币销售、有些用法币销售吗？**
可以。支付通道是在发布页面按商品逐一选择的，而不是按账户，所以同一个卖家资料下可以让两条通道并行运作。两条通道的手续费同为 10%。

**法币和加密货币的手续费不一样吗？**
不会。两者的手续费完全相同：都是固定的 10% 平台手续费，在代码中只定义了一处，无论买家用哪种方式付款都同样适用。

**我实际多久能收到钱？**
加密货币收款是即时的，在购买那笔交易中同步完成，直接进入你的钱包。法币收款会在销售发生时立即进入你的 Stripe Express 余额，随后按照 Stripe 自己的标准打款周期打入你的银行账户，具体时长取决于你所在的国家和账户状态。

**代币化商品可以用法币销售吗？**
不可以。代币化发布始终以加密货币结算。法币（Stripe）只适用于直接的、非代币化的付费商品。

**我能卖什么？**
智能体、提示词和工具都可以作为付费商品上架。根据你构建的内容能为买家节省多少时间或专业知识来定价即可。

## 开始行动

如果你已经准备好开始销售，按以下顺序操作：

1. **设置收款方式（如果你想支持银行卡）：** 打开[卖家标签页](https://swarms.world/platform/account?tab=seller)，选择你所在的国家，点击 Set up seller account。
2. **发布你的商品：** 前往 [/launch](https://swarms.world/launch)，将你的智能体、提示词或工具标记为 Paid，设定价格，并选择支付通道。
3. **看看已经有哪些商品：** 浏览[Marketplace 注册表](https://swarms.world/platform/registry)，在发布之前，先看看类似商品是怎么定价和描述的。

发布不花一分钱，前面也没有任何门槛需要跨越，而且你能拿到每笔销售额的 90%。剩下的，就是构建一件真正值得购买的东西。
