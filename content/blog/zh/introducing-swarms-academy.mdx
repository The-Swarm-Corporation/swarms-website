# Swarms Academy 正式上线：多智能体技术栈免费课程

今天，我们正式发布 [Swarms Academy](https://swarms.ai/academy)，一个覆盖 Swarms 生态全部产品的免费学习中心。学院首发一门 Swarms API 旗舰课程，同时提供 Swarms Marketplace、开源 Swarms 框架以及整个技术栈其他产品的学习路径。

我们的目标很简单：让一位从未接触过 AI 智能体的开发者，一节课一节课地成长为能够构建生产级多智能体系统的工程师，并且不收取任何费用。

## Swarms API 课程

本次发布的核心是 [Swarms API 课程](https://swarms.ai/academy/swarms-api)，一套直接基于官方 API 文档构建的四部分课程体系。课程包含 31 节课，完成全部内容约需六小时，唯一的前置要求是基础的 Python 知识。

四个部分层层递进：

**第一部分：基础入门。** 了解 Swarms API 是什么、身份认证如何工作，以及一个智能体的完整结构。你将运行健康检查，通过 `POST /v1/agent/completions` 执行你的第一次智能体补全，探索可用模型，并在进入下一部分之前学会解读 `usage` 字段和所有常见错误状态码。

**第二部分：单智能体能力。** 一个智能体能做的远不止回答提示词。这一部分涵盖多轮对话历史、实时流式输出、基于模式约束的结构化输出、函数工具、网络搜索、用于连接外部系统的 MCP 集成、将 `max_loops` 设为 auto 的自主执行模式，以及可作为现有 SDK 代码即插即用替代方案的 OpenAI 兼容端点。

**第三部分：多智能体编排。** 整门课程的核心。API 支持的每一种编排架构都配有可运行的示例：SequentialWorkflow 流水线、ConcurrentWorkflow 并行扇出、带有指挥者的 HierarchicalSwarm 团队、GroupChat、MixtureOfAgents、MajorityVoting、辩论与评审团模式、AgentRearrange 自定义流程，以及用于有向无环图的 GraphWorkflow。这一部分最后提供一份实用指南，帮助你为具体任务选择合适的架构。

**第四部分：生产落地。** 可用的演示与生产系统之间的差距在于工程运维。这一部分涵盖大规模批量处理、面向高难度分析问题的推理智能体、速率限制与重试策略、通过模型分层与夜间调度实现的成本优化、基于日志与指标端点的可观测性，以及一份上线前可以逐项核对的生产就绪清单。

## 在课程页面内直接运行真实 API 请求

只读文档远不如亲手调用 API。课程在课节中内嵌了十二个实时 API 试验。只需粘贴一次你的 API 密钥（密钥仅保存在你的浏览器中，且只发送至 `api.swarms.world`），即可在页面内直接执行真实请求：运行健康检查、发起你的第一次智能体补全、启动一个双智能体顺序蜂群、查看你自己的速率限制。

每个 POST 试验都带有可编辑的 JSON 请求体。修改任务内容、改写系统提示词、调整温度参数，然后再次运行。每个试验都会展示状态码、延迟和完整响应，并明确标注是否消耗积分额度。当你想切换到自己的终端时，一键复制即可生成等效的 cURL 命令。

## 积分、等级与徽章

课程进度保存在你的浏览器中，无需注册账号。完成每项学习活动都会获得积分：

- 每完成一节课获得 10 积分
- 每成功运行一个实时 API 试验获得 15 积分
- 每完成一个检查点项目获得 25 积分
- 测验每答对一题获得 5 积分，满分额外奖励 10 积分

积分不断累积，等级随之提升：从新兵（Recruit）到智能体操作员（Agent Operator）、蜂群构建者（Swarm Builder）、编排师（Orchestrator），直至蜂群架构师（Swarm Architect）。完成某一部分的全部课程、检查点和测验即可获得该部分的徽章。课程页面上的进度面板会展示你的等级、各部分完成度，以及一个可以随时回到上次学习位置的继续按钮。

## 检查点与测验

每个部分以两项内容收尾：一个检查点项目和一次知识测验。

检查点项目是覆盖该部分全部知识的实战构建。你将在第一部分构建一个可用的研究智能体，在第二部分构建一个结构化研究助手，在第三部分用三种不同方式构建同一份投资备忘录（以便横向比较各架构的成本与质量），并在第四部分完成一个带成本核验的夜间批量流水线作为毕业项目。

测验为评分式多选题，全课程共 26 道题，每道题都附有正确答案的解析。只有你的最好成绩计入积分，因此重考提分永远值得。
