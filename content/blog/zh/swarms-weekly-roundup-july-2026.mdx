# Swarms 每周回顾 [7 月 13-20 日]：Swarms Marketplace 法币支付、7,000 GitHub Star、Swarms Academy

过去一周，Swarms 技术栈几乎每一层都有进展：教育、开源框架、Marketplace 支付通道、云平台，以及支撑这一切的长期思路。本文将这五项更新汇总于一处，为每一项提供简短说明，并附上对应完整公告的链接。

## Swarms Academy 正式上线

![Swarms Academy](/academy_picture.png)

我们发布了 Swarms Academy，一个面向构建生产级 AI 智能体与多智能体系统开发者的免费学习平台。首发课程是 Swarms API 课程，包含四个以生产为核心的模块、共 31 节课程。课程覆盖完整工作流：从首次完成身份验证的 API 请求，到流式输出、结构化输出、工具调用、MCP 集成等单智能体能力，再到多智能体编排与生产环境相关议题。课程配有可在页面内直接运行的真实 API 练习、实战项目、测验和进度跟踪，开发者可以从第一次智能体请求一路走到部署完整的多智能体系统。📚

阅读完整公告：[Swarms Academy 正式上线](/blog/introducing-swarms-academy)

## Swarms 达成 7,000 GitHub Star

![Swarms 达成 7,000 GitHub Star](https://pbs.twimg.com/media/HNdR91uXoAAwmHK?format=jpg&name=large)

在作为研究项目起步四年之后，Swarms 在 GitHub 上突破了 7,000 Star。这段时间里，它已成长为一个被全球数千名开发者用于生产环境构建多智能体系统的开源平台。这一里程碑归功于四年开发中提交 issue、发起 pull request、共同塑造框架的贡献者、用户与研究者。它标志着项目走向成熟，而非终点。

阅读完整公告：[Swarms 达成 7,000 GitHub Star](/blog/swarms-7000-github-stars)

## Swarms Marketplace 法币支付上线

![由 Stripe 驱动的 Swarms Marketplace 法币支付上线](/swarms_blog_fiat_cover.png)

由 Stripe 驱动，Swarms Marketplace 现已支持法币支付。超过 100 个国家的买家可以使用银行卡、Apple Pay、Google Pay、支付宝、Klarna 等熟悉的方式，跨 100 多种货币购买 AI 智能体和提示词。系统基于 Stripe Connect 构建：卖家只需连接一次账户，即可获得自动打款至银行账户，并在同样的 10% 平台费下保留每笔销售的 90%。买家付款后即刻获得终身访问权限，无需钱包或手动提现步骤。原有的加密支付支持继续完整保留，两种方式可在同一个卖家账户下并行运行。

阅读完整公告：[智能体经济现已支持刷卡支付：由 Stripe 驱动](/blog/fiat-payments-live-swarms-marketplace)

## Swarms Cloud 两个月更新汇总

![Swarms Cloud 更新日志：两个月的平台迭代](/swarms_cloud_2month.png)

我们发布了一份完整的更新日志，涵盖过去两个月在 Swarms Cloud 上交付的全部内容。它没有只挑选重大发布，而是以逐日时间线记录了进入生产环境的每一项功能、改进、性能优化和缺陷修复。重点包括：用于可视化编排多智能体系统的全新 Workflow Builder、收录超过 1,500 个模型的公开模型目录、提供实时成本与用量分析的 Token Usage 仪表盘、为新用户提供 $5 免费 API 额度的引导式上手流程，以及数十项可靠性增强与界面改进。

阅读完整日志：[Swarms Cloud 更新日志：两个月的平台迭代](/blog/swarms-cloud-changelog)

## 自主经济：创始人主题演讲

![创始人在 AI Salon 发表关于自主经济的主题演讲](/autonomous_economy_keynote.png)

Swarms 联合创始人兼 CEO Kye Gomez 在 AI Salon 面向 80 多位观众发表主题演讲，探讨商业的未来：主要由 AI 智能体运营的公司，通过软件而非增加人手来实现规模化。核心观点是：随着智能成本下降，一种新的运营模式变得可行，公司可以以极低的管理开销持续运转，并通过编排智能系统而非招聘来扩展能力。演讲将 Swarms 所构建的基础设施定位为这一转变的底座。

阅读完整公告：[自主经济时代即将到来：创始人在 AI Salon 的主题演讲](/blog/autonomous-economy-keynote)

## 结语

综合来看，这些更新勾勒出同一条主线。Swarms Academy 降低了学习技术栈的门槛，7,000 Star 里程碑体现了框架四年的成熟度，法币支付向全球买家与卖家开放了 Marketplace，Cloud 更新日志记录了平台的稳步推进，创始人主题演讲则指明了后续工作的方向。每一部分都服务于同一个目标：让构建、部署和扩展生产级多智能体系统变得切实可行。我们才刚刚开始。
