# 代币化智能体：是什么、如何发布，以及创作者如何获得持续收益

大多数市场只卖一次拷贝。买家付一次钱，获得访问权限,交易就此结束。这种模式行得通,但也把创作者的收益上限锁死在他们能促成的销售笔数上。代币化是 [Swarms Marketplace](https://swarms.world) 提供的另一种经济模型:与其出售一份固定价格的拷贝,你把自己的智能体或提示词变成 Solana 上可交易的资产,只要这份资产还在被交易,你就能持续从每一笔交易中抽成。

本文将讲清楚:代币化智能体到底是什么,背后的债券曲线如何运作,创作者具体如何获得报酬,以及如何自己动手发布一个。

## 什么是代币化智能体?

代币化智能体(或代币化提示词)本质上是一个普通的市场listing,只是多了一个属性:它在 Solana 上拥有属于自己的代币。当你在 swarms.world 发布一个智能体或提示词时,代币化只是同一个发布流程里的一个开关,就在「免费」和「付费」定价选项旁边。工具(Tools)不支持代币化,这个开关对该内容类型是禁用的。

打开这个开关后,会新增两个必填字段:

- **Ticker Name(代码符号)。** 一个最多 10 个字符的大写符号,只能是字母和数字,例如 `RESEARCHBOT` 或 `MEDPROMPT`。发布页面还能根据你 listing 的名称和描述,帮你自动生成候选代码符号。
- **Quote Mint(计价币种)。** 你的代币在债券曲线上用来计价交易的货币:SOL 或 USDC。

注意这份清单里缺了什么:没有总供应量字段,没有起始价格字段,也没有需要设置的代币数量。这些都是固定的平台默认值,而不是创作者可以自行决定的参数,因为代币化的本意不是让创作者手动调参做一次融资,而是一个标准化、可重复使用的封装,把任何智能体或提示词变成一个规则始终一致的交易对。

最终结果是:一个 listing 同时是市场上的产品,也是一个实时可交易的代币。人们仍然可以像浏览其他智能体或提示词一样发现它、评价它、使用它,同时也能在链上买卖它的代币,而每一笔这样的交易都会给创作者带来收益。

## 债券曲线如何运作

在底层,代币化通过 Solana 上一种名为动态债券曲线(Dynamic Bonding Curve)的自动做市机制运行,由 Jupiter 的发布基础设施提供支持。你完全不需要理解债券曲线的数学原理就能使用这个功能,但了解它的大致形状,能帮你更好地判断作为创作者应该预期什么。

债券曲线是一个定价公式,而不是一本订单簿。你的交易不需要在对手方等到一个匹配的买家或卖家:曲线本身随时准备好按照当前流通量决定的价格进行买卖。早期买家能拿到更低的价格,随着代币被买入得越多,价格会沿着曲线自动上涨。

Swarms 上所有代币化 listing 都以相同的两个市值节点启动,它们是平台层面的固定常量:

| 节点 | 目标市值 |
| --- | --- |
| 初始市值 | 约 3,246 美元 |
| 迁移市值 | 约 46,468 美元 |

对于以 SOL 计价的代币,这些美元目标会在你发布的那一刻,按当时的实时汇率换算成 SOL,这样无论当天 SOL 的价格是多少,曲线的起点始终对应一个一致的真实价值。以 USDC 计价的代币则直接使用美元数字。「迁移」指的是代币交易量累积到一定程度后,从债券曲线毕业,进入标准流动性池的那个节点,这也是 Solana 上各类发布平台普遍采用的毕业模式。

发布一个代币需要支付一次性的 0.04 SOL 费用,这笔费用会在发布流程中提前展示给你,用于覆盖部署代币及其资金池的链上成本。发布页面明确说明了一个重要细节:创建代币并不会自动把任何代币分配给你。如果你想持有自己的代币,无论是出于所有权考虑、用于门槛控制,还是单纯想成为它的第一个持有者,你都需要在代币上线后尽快自己买入,和其他任何人一样。

## 如何发布代币化智能体

整个流程都在[发布页面](https://swarms.world/launch)完成,相比普通发布只多出几个步骤:

1. **像平时一样填写你的 listing**:名称、描述、代码或提示词内容,以及发布一个标准智能体或提示词所需的其他一切。
2. **在「定价」部分打开代币化开关**。仅对智能体和提示词开放,不支持工具。
3. **设置 Ticker Name**,自己输入或从自动生成助手给出的建议中挑选。
4. **选择 Quote Mint**:SOL 或 USDC。
5. **决定是否启用 Frenzy 模式和 Vault 模式**,两者都是可选项,详见下文。
6. **连接一个 Solana 钱包。** 代币化要求你在发布前连接钱包,因为代币的铸造和部署是同一流程的一部分。
7. **发布。** listing 立即上线,代币也在同一个操作中完成链上部署。系统提示时支付 0.04 SOL 费用,如果你想持有自己的代币,记得发布后尽快买入。

从这一刻起,你的智能体或提示词就是一个普通的市场 listing,只是附带了一个实时可交易的代币。

## 创作者如何赚钱:手续费分成

这正是让代币化区别于一次性销售的核心机制。债券曲线上每一笔买入和卖出交易都会产生 1.0% 的总交易手续费,这笔费用会自动分成:

| 手续费接收方 | 分成比例 |
| --- | --- |
| 创作者 | 0.5% |
| Swarms 平台 | 0.5% |
| **总交易手续费** | **1.0%** |

这 0.5% 不是一次性付款。只要代币还有交易量,这个比例就会适用于每一笔交易,没有期限。发布了一个受欢迎智能体的创作者,不是只在发布那一刻拿到报酬,而是此后每次有人买卖这个代币,他们都能获得收益。这正是与传统市场销售的核心区别:固定价格的 listing 每个买家只付一次钱,上限就是买家的总数。而代币化 listing 让创作者持续从交易量中抽成,不管这些交易量来自同一批持有人反复买卖、来自价格变动后进场的新买家,还是来自与传统意义上「一次销售」毫无关系的其他市场活动。

手续费会随着交易发生实时在链上累积。要收取这些费用,需要在你的账户设置中的 Creator Fees(创作者手续费)标签页连接 Solana 钱包,在那里你可以看到每个代币未领取和已领取的手续费,并通过一笔由你亲自签名的交易完成领取。

### Frenzy 模式:让创作者分成翻倍

在发布时,你可以选择为智能体和提示词开启 Frenzy 模式。开启后会改变分成比例,让创作者获得所有买卖交易量的 1%,是标准 0.5% 比例的两倍。这是一笔直接的权衡:如果你有信心自己的 listing 会产生真实的交易活跃度,Frenzy 模式能从第一天起就让你在每笔交易中的收益翻倍。

### Vault 模式:把代币变成访问门槛

Vault 模式是另一个独立的可选开关,它改变了代币的用途。代币不再只是单纯的交易工具,Vault 模式会把智能体或提示词本身设为门槛,只有持币者才能使用。用户需要在自己连接的钱包中持有价值大约 1 美元的该代币,才能解锁被锁定的内容。

由于 Vault 模式已经通过代币持有量来控制访问权限,它无法与单独的美元付费定价共存:如果你在一个设为「付费」的 listing 上开启 Vault 模式,发布页面会自动把定价切回「免费」,因为在持币门槛之上再收取美元价格是多余的,代币本身已经充当了付费墙。

选择 Vault 模式的战略意义在于它所创造的买入压力。任何想使用你的智能体或提示词的人,都必须先去获取你的代币,哪怕只需要买入很小的一笔以达到持有门槛,这是普通付费 listing 所不具备的结构性需求驱动力。

## 代币化与普通付费 Listing 的对比

| | 普通付费 Listing | 代币化 Listing |
| --- | --- | --- |
| 创作者如何获得收益 | 每个买家一次性支付固定美元价格 | 从每一笔交易中持续抽成 |
| 价格发现方式 | 由创作者手动设定 | 由债券曲线决定 |
| 所有权 | 买家获得内容使用许可 | 买家持有可交易的链上代币 |
| 平台抽成 | 每笔销售的一定比例 | 每笔交易的 0.5% |
| 访问控制方式 | 固定价格的付费门槛 | 基于持币门槛的 Vault 模式 |
| 适用范围 | 智能体、提示词和工具 | 仅限智能体和提示词 |

两种模式没有绝对的优劣之分。固定价格的 listing 更简单、更可预测,适合价值明确稳定的内容。代币化则适合那些人们愿意去交易、讨论的内容,在这种情况下,持续的交易量,而不是一次性的购买,才是真正的收益来源。

## 常见问题

**我可以把工具(Tool)代币化吗?**
不可以。代币化仅适用于智能体和提示词。工具不支持这项功能,发布工具时代币化开关是禁用的。

**创建代币后,我会自动拥有自己的代币吗?**
不会。创建代币并不会自动分配任何代币给你。如果你想持有,需要在发布后尽快自己买入,和其他买家一样。

**交易的具体手续费分成是多少?**
每笔交易总计 1.0%:默认情况下创作者获得 0.5%,Swarms 平台获得 0.5%。开启 Frenzy 模式后,创作者的分成会翻倍,达到所有买卖交易量的 1%。

**我可以同时使用 Vault 模式和美元付费定价吗?**
不可以。Vault 模式已经通过持币量控制访问权限,因此无法与美元付费定价共存。在付费 listing 上开启 Vault 模式,系统会自动把定价切回「免费」。

**代币化 listing 需要加密钱包吗?**
需要。代币化要求你在发布前连接 Solana 钱包,因为代币的铸造和部署是发布流程的一部分,在链上完成。

**作为创作者,我该如何领取自己赚到的手续费?**
在账户设置的 Creator Fees(创作者手续费)标签页连接你的钱包。在那里你可以看到每个代币未领取和已领取的手续费,并通过一笔由你亲自签名的交易完成领取。

**代币可以用哪些货币计价?**
SOL 或 USDC。你在配置代币化时选择 Quote Mint,它决定了债券曲线用哪种货币计价。

## 开始使用

代币化能把你的智能体或提示词,从一次性销售变成与真实交易活动挂钩的持续收益来源。如果这种模式适合你正在构建的东西,可以从这里开始:

- **发布一个代币化智能体或提示词:** 前往[发布页面](https://swarms.world/launch),填写你的 listing,打开代币化开关,设置好代码符号和计价币种。
- **看看已经上线的项目:** 在[市场注册表](https://swarms.world/platform/registry)中浏览代币化 listing 以及其他所有内容。
- **深入了解技术细节:** 代币化的完整技术说明记录在 [docs.swarms.ai](https://docs.swarms.ai/docs/marketplace/tokenization_details)。

如果你构建的东西是人们愿意长期持有、而不只是用一次就走的,代币化就是让你每次交易都能获得回报的方式。
