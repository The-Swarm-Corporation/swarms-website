# 我们是谁：Swarms 公司与构建集体超级智能的使命

Swarms 公司是一家研究与工程公司，只有一个使命：构建集体超级智能。我们沿着一条连贯的路径来实现它，从对多智能体系统的研究，到将这些系统部署进真实经济，让它们完成真实的工作、创造真实的价值。

这篇文章是对我们是谁、我们构建什么、以及我们为何如此的直白陈述。

## 我们下的赌注

整个 AI 行业都围绕着一个理念组织起来：智能以单一心智的形式扩展。把模型做得更大、训练得更久，你最终会得到一个比任何人类都更聪明的人工心智。

我们认为这个框架搞错了衡量单位。人类观察到的最强大的智能都不是个体，而是集体。没有任何一个人独自设计了现代经济、发现了科学知识的总和，或建造了互联网。在一切重要的时间尺度上，文明都胜过天才。

对机器而言同样如此。**集体超级智能（CSI）** 是一个由专业化智能体组成的网络，它们相互推理、协调、纠错，而智能既存在于任何单个智能体之中，也同样存在于拓扑结构之中。我们相信 CSI 在能力和经济价值上都将超越单一的庞大模型，而构建它所需的基本要素今天就已经存在。完整论证请见[《集体超级智能：为何 CSI 将超越 AGI 与 ASI》](/blog/collective-superintelligence)。

## 我们做什么：先研究，再部署

我们的使命分为两个彼此滋养的部分。

**我们研究多智能体系统。** CSI 中的难题不只是让单个智能体更聪明，而是关于编排：智能体如何分工，如何通过层级、辩论和流水线进行协调，如何共享记忆，如何相互验证，以及一个智能体网络如何在规模化下保持可靠。正是这些研究，把一堆智能体变成一个连贯的智能体。

**我们把它们部署进经济。** 停留在论文里的研究改变不了任何东西。我们构建基础设施，让多智能体系统投入生产任务：分析、自动化、编程、运营，以及支撑一家现代企业运转的大量长尾认知劳动。每一次部署都是对研究的检验，每一个结果都反哺下一次迭代。

从研究到部署再回到研究，这个循环就是公司的全部。

## 我们构建什么

这一使命通过一整套产品来表达，每一层都把 CSI 从一个理念变成开发者今天就能运行的东西。

- **Swarms 框架。** 构建多智能体系统的开源基础，被数千名开发者使用，如今已突破 7,000 个 GitHub 星标。它提供了组装 CSI 所需的智能体、编排拓扑与协调基元。
- **Swarms API。** 位于 [swarms.ai](https://swarms.ai) 的单一端点，用于定义智能体、把它们组合成蜂群、并选择编排架构，调度、状态与计费都已为你处理妥当。一个部门就此变成一个配置文件。
- **Swarms 市场。** 位于 [swarms.world](https://swarms.world) 的全球市场，开发者在此发现、发布并变现智能体、提示词与工具。CSI 的组件由此得以构建一次、处处复用，创作者从每一笔销售中获得收益。
- **Swarms Academy。** 一个免费学习平台，带开发者从第一次经过身份验证的请求，走到部署一套多智能体系统。

每一层都服务于同一个终点。框架让 CSI 可构建，API 让它可运行，市场让它成为一个经济体，而 Academy 把更多的构建者带入其中。

## 为何经济是检验场

我们以完成的真实工作来衡量进展，而不只是基准分数。企业本身就是信息处理系统，其中几乎每一项职能，从研究到销售再到运营，都可以表达为一支协同工作的智能体团队。随着认知劳动的成本下降，掌握编排的组织将以持续运转的蜂群取代僵硬的层级结构。

我们创始人的主题演讲阐明了这将通向何方：先是自主部门，然后是自主公司，人类则转向对完成工作的系统的所有权。完整论证见[《自主经济》](/blog/autonomous-economy-keynote)。

## 我们要去往何方

四年前我们从一个研究项目起步，成长为一个开源平台、一个 API、一个市场，以及围绕这三者形成的经济体。从这里出发的方向与我们的起点一脉相承：对智能体如何协调进行更深入的研究，以及对这些研究成果进行更广泛的部署。

如果你愿意与我们一起构建智能的未来，请从 [swarms.ai](https://swarms.ai) 开始，获取[免费 API 密钥](https://cloud.swarms.world/api-keys)，并在 [docs.swarms.ai](https://docs.swarms.ai) 阅读研究成果。集体超级智能不是一件需要等待的事。我们正在构建它，而大门已经敞开。
